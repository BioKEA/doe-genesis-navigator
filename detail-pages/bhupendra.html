Introduction

Computational environmental scientist specializing in remote sensing and other atmospheric data-driven analysis of clouds, and precipitation. My work combines weather radar, satellite, and reanalysis data with statistical methods and machine learning to study convective initiation, storm evolution, cloud microphysics, and rainfall processes from  cloud to planetary scales.

I develop algorithms for radar echo classification, storm tracking, convective–stratiform separation, and high-resolution rainfall downscaling. Recently I have been exploring self-supervised and representation learning methods for large multi-sensor atmospheric datasets.

Interested in collaborations that integrate atmospheric observations, AI/ML, and large Earth-system datasets to improve understanding of convection, precipitation processes, and land–atmosphere interactions.

Relevant Projects

One major project involved development of the SAGE Edge Computing system for atmospheric observations. The work focused on designing efficient algorithms for real-time analysis of remote sensing data at the network edge. The scope included implementing cloud motion estimation methods using phase correlation and optical flow to enable low-latency processing of large atmospheric image streams. This work demonstrated that computationally intensive atmospheric analysis tasks can be deployed on distributed edge hardware, enabling scalable and energy-efficient processing for observational networks.

Another project involved radar data analysis using observations from the Atmospheric Radiation Measurement (ARM) user facility to study convective structure and precipitation processes. The work developed algorithms for hydrometeor classification, storm identification, and tracking of convective systems from volumetric radar data, enabling extraction of dynamical and microphysical information from radar reflectivity and supporting studies of convective organization and rainfall variability.
